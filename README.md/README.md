# Portfolio Evaluation App

A Python application to evaluate user portfolios using financial metrics such as Sharpe ratio.  
The app provides advice, portfolio optimization, and generates reports for users.

---

## Features

- Add and manage multiple user portfolios
- Calculate key financial metrics (Sharpe ratio)
- Generate actionable advice for portfolios
- Optimize portfolio weights for risk-adjusted returns
- Generate text-based reports
- Simple CLI interface
- Modular architecture (services, routes, utils)
- Secure password handling with bcrypt
